# Future_DS_3

📊 Marketing Funnel Dashboard
📌 Overview
This project analyzes the customer journey from visitors to purchases using a marketing funnel dashboard. It highlights campaign performance, conversion efficiency, and revenue insights.
🎯 Objectives
Analyze funnel performance
Track user journey
Identify drop-offs
Compare campaigns
Improve conversion & ROI
📊 Key Metrics
Visitors: 41,000
Leads: 11,350
Purchases: 3,260
Revenue: 475,000
Conversion Rate: 7.95%
📈 Insights
Top campaigns drive higher revenue
Drop-offs exist at key funnel stages
Conversion varies by campaign
Some campaigns deliver better ROI
